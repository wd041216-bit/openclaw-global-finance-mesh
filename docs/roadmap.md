# Roadmap

## Phase 1

- Ship Pack validation
- Ship deterministic decision packets
- Ship evidence graph snapshots
- Ship replay summaries

## Phase 2

- Add connector adapters for ERP and ticketing systems
- Add pack registry and version promotion workflow
- Add retention controls and backup discipline on top of the SQLite audit ledger
- Add off-box backup automation for audit and auth session recovery

## Phase 3

- Add country-specific regulatory bundles with official-source references
- Add approval integration with real workflow systems
- Add output artifact generation for vouchers and filing drafts
- Add group mapping or SCIM-compatible identity lifecycle

## Phase 4

- Add canary publishing and rollback automation
- Add KPI dashboards for auto-execution and review queues
- Add graph database export for complex trace analysis
- Add immutable archival storage and stronger tenant/environment isolation
